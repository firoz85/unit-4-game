Crystal Collector Game

How the Game Works:

  • At the start of the game - given a random number!
  • Each of the 4 crystals below have been assigned a hidden number.
  • By clicking on a crystal you will add a specific amount of points to "Your Crystal Count".
  • Only WIN the game by matching when "Your Crystal Count" = "Random Crystal Number".
  • If Crystal Count is higher than the random number - LOSE the game.
  • Each time the game restarts - value of each crystal are never the same. Good Luck!
